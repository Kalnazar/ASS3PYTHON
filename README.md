## Usage

* Import libraries
* Generate and set SECRET_KEY
* Set SQLALCHEMY_DATABASE_URI as postgreSQL or sqlite

```
>>> from flaskblog import db
>>> from flaskblog.models import Users
>>> db.create_all()
```
